🚀 Como abrir

Extraia o arquivo .zip em uma pasta
Abra a pasta no VS Code: File → Open Folder
Instale a extensão Live Server (ritwickdey.liveserver)
Clique com o botão direito em index.html → "Open with Live Server"
O site abre automaticamente no navegador!


Também funciona abrindo o index.html diretamente no navegador (duplo clique).

✨ Funcionalidades JavaScript

Botão "Voltar ao topo" — aparece após rolar 300px, some ao chegar ao topo
Link ativo na navbar — destaque automático conforme a seção visível
Sombra na navbar — aparece ao rolar a página
Barras de habilidade animadas — preenchem ao entrar na tela (IntersectionObserver)
Formulário com validação — usa a API nativa do Bootstrap + HTML5
Animação de entrada dos cards — fade-in escalonado ao rolar
